# Sales Data Analysis

This is my Project 1 for the Cognetix Technology internship (Foundational Stage).

## What this project does
- Loads sales data from a CSV file
- Cleans the data (removes nulls and duplicates)
- Calculates total, average, max and min sales
- Creates bar charts and pie charts for category and region wise sales
- Shows top 10 products and countries by sales

## Dataset
Downloaded from Kaggle: https://www.kaggle.com/datasets/kyanyoga/sample-sales-data

## Libraries used
- pandas
- numpy
- matplotlib
- seaborn

## How to run

Install the libraries:
```
pip install pandas numpy matplotlib seaborn jupyter
```

Place the CSV file in the same folder and open the notebook:
```
jupyter notebook analysis.ipynb
```

## Files
- `analysis.ipynb` - main jupyter notebook
- `sales_data_sample.csv` - dataset (download from kaggle)
